### 2. 多态的四大版图

为了让你更清晰地定位运算符重载的位置，我们可以把 C++ 中的多态分为两个维度：

#### **A. 编译时多态（静态多态 / Static Polymorphism）**

在代码**编译阶段**，编译器就已经知道要调用哪个函数了。

- **重载多态 (Ad-hoc Polymorphism)**：包括函数重载和**运算符重载**。
- **参数化多态 (Parametric Polymorphism)**：主要是 **模板（Templates）**。

#### **B. 运行时多态（动态多态 / Dynamic Polymorphism）**

在程序**运行阶段**，根据对象的实际类型（通过虚函数表 Vtable）来决定调用哪个函数。

- **包含多态 (Inclusion Polymorphism)**：即 **虚函数（Virtual Functions）** 和继承。
- **强制多态 (Coercion Polymorphism)**：隐式类型转换（如将 `char` 传给 `int` 参数的函数）。
