- 如果一个**类**中重载了**函数调用运算符**，那么这个类的对象被称为**函数对象**
- **函数对象**是一个**类**，而不是一个**函数**

#### 1.函数对象的基本使用

- [x] 函数对象在使用时，可以像普通函数那样调用，可以有参数和返回值

```c++
# include <iostream>
using namespace std;

class MyAdd
{
public:
	int operator()(int a, int b)
	{
		return a + b;
	}
};

int main()
{
	MyAdd my_add;
	cout << my_add(1, 5) << endl;

	return 0;
}
```

- [x] 由于函数对象是一个类，所以函数对象可以拥有自己的状态

```c++
# include <iostream>
using namespace std;

class MyAdd
{
public:
	int count;
	MyAdd() :count(0){}

	int operator()(int a, int b)
	{
        //通过类内的成员变量count来统计my_add的调用次数，普通的函数做不到这一点
		count++;
		return a + b;
	}
};

int main()
{
	MyAdd my_add;
	cout << my_add(1, 5) << endl;    //6
	my_add(2, 4);
	my_add(3, 2);
	cout << "my_add一共执行了" << my_add.count << "次" << endl;   //3次

	return 0;
}
```

- [x] 由于函数对象是一个类，所以可以作为其他函数的参数

```c++
# include <iostream>
using namespace std;

class MyAdd
{
public:
	int count;
	MyAdd() :count(0){}

	int operator()(int a, int b)
	{
		count++;
		return a + b;
	}
};

//my_add作为其他函数的参数，这里虽然看起来像脱裤子放屁，但是只是做个演示
int doAdd(MyAdd& my_add, int a, int b)
{
	int add = my_add(a, b);
	return add;
}

int main()
{
	MyAdd my_add;
	cout << my_add(1, 5) << endl; //6
	my_add(2, 4);
	my_add(3, 2);
	cout << doAdd(my_add, 10, 20) << endl; //30
	cout << "my_add一共执行了" << my_add.count << "次" << endl; //4

	return 0;
}
```

#### 2.谓词predicate

- 谓词是一个返回**布尔值**的逻辑判断器，可以是**普通函数**，**函数对象**和**lambda表达式**
- 谓词根据参数的数量分为**一元谓词**与**二元谓词**

- [x] 普通函数判断一个int是否大于5

```c++
# include <iostream>
using namespace std;

bool bigger5(int a)
{
	if (a > 5)
	{
		return true;
	}
	else
	{
		return false;
	}
}

int main()
{
	int a = 10;
	if (bigger5(a))
	{
		cout << "a比5大" << endl;
	}
	else
	{
		cout << "a比5小" << endl;
	}
	return 0;
}
```

##### 一元谓词

`std::find_if`   通过自定义条件进行查询，三个参数分别为（迭代器，迭代器，谓词），返回值为迭代器。
默认情况下，判断输入值是否等于某个值，如果等于则返回true

-  由于这里的谓词是做函数参数的，所以普通函数谓词是不行的，必须用函数对象谓词

```c++
# include <iostream>
# include <vector>
# include <algorithm>
using namespace std;

class Greater5
{
public:
	bool operator() (int a)
	{
		if (a > 5)
		{
			return true;
		}
		else
		{
			return false;
		}
	}
};

int main()
{
	vector<int> a;
	//给a填了十个值
	for (int i = 0; i < 10; i++)
	{
		a.push_back(i);
	}

	Greater5 greater5;
	//find_if的返回值为迭代器
	//这里的参数greater5虽然并没有带上重载的()，但是find_if自动地把迭代器指向的元素传给了greater5(int a)
	vector<int>::iterator it = find_if(a.begin(), a.end(), greater5);
	if (it == a.end())
	{
		cout << "没有找到满足条件的元素" << endl;
	}
	else
	{
		cout << "找到了满足条件的元素，且该元素的值为" << *it << endl;
 	}
	
	return 0;
}
```

我们使用Greater5类判断一个数a是否大于5，然后把这个谓词传给find_if做参数，就可以让find_if查询大于5的元素

##### 二元谓词

`std::sort`   从小到大排序，有无谓词和有谓词两种版本
默认情况下，判断输入的val1<val2?如果小于则返回true

```c++
# include <iostream>
# include <algorithm>
# include <vector>

//输出vector的函数
void print_vec(std::vector<int>& a)
{
	for (auto i = a.begin(); i != a.end(); i++)
	{
		std::cout << *i << std::endl;         
	}
}

class MyCompare
{
public:
    //把sort中val1<val2的判断变成val1>val2
	bool operator()(int val1, int val2)
	{
		if (val1 > val2)
		{
			return true;
		}
		else
		{
			return false;
		}
	}
};

int main()
{
	std::vector<int> a;
	for (int i = 5; i >= 1; i--)
	{
		a.push_back(i * 10);
	}
	print_vec(a);   //50, 40, 30, 20, 10

	std::sort(a.begin(), a.end());    
	print_vec(a);     //在sort后，变成了从小到大排列；10,20,30,40,50

	std::sort(a.begin(), a.end(), MyCompare());    //匿名函数对象，不需要单独创建一个对象
	print_vec(a);     //在自定排序规则后，变成了从大到小排列：50,40,30,20,10

	return 0;
}
```

##### 算数仿函数

以取反`negate`和加法`plus`为例，negate是一元的，plus是二元的，具有代表性

```c++
# include <iostream>
# include <functional>

void test1()
{
    //类模板创建对象
	std::negate<int> n;
	std::cout << n(20) << std::endl;          //-20
}

void test2()
{
    //虽然plus有两个参数，但是只需要一个int,因为在定义中，两个参数的类型都是T
	std::plus<int> p;
	std::cout << p(10, 20) << std::endl;    //30
}

int main()
{
	test1();
	test2();
	return 0;
}
```

##### 关系仿函数

返回值为bool，以大于`greater`为例

```c++
#include <iostream>
#include <algorithm>
#include <functional>
#include <vector>

void print_vec(std::vector<int>& a)
{
	for (auto it = a.begin(); it != a.end(); it++)
	{
		std::cout << *it << std::endl;
	}
}

int main()
{
	std::vector<int> a;
	for (int i = 1; i < 6; i++)
	{
		a.push_back(i * 10);
	}
    //greater和我们自己写的谓词是一样的，这里greater<int>实例化成类之后再进行匿名调用
	std::sort(a.begin(), a.end(), std::greater<int>());
	print_vec(a);

	return 0;
}
```
