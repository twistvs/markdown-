- 运算符重载属于多态中的静态多态中的重载多态

 **成员函数重载 vs. 全局函数重载**

| **重载方式**         | **定义位置**        | **调用逻辑**               | **典型用途**                                          |
| -------------------- | ------------------- | -------------------------- | ----------------------------------------------------- |
| **类内（成员函数）** | 在 `class` 内部定义 | 左操作数必须是该类的对象   | 赋值 (`=`)、下标 (`[]`)、调用 (`()`)、成员访问 (`->`) |
| **类外（全局函数）** | 在类外部定义        | 可以灵活处理左右操作数顺序 | 算术运算 (`+`, `-`)、流输入输出 (`<<`, `>>`)          |

#### 1.加号运算符重载

```c++
# include <iostream>
# include <array>
using namespace std;

class Person
{
public:
	Person() {}
	Person(int a, int b) :a(a), b(b) {}

	//类内的加号重载
	Person operator+ (const Person& p)
	{
		Person temp;
		temp.a = p.a + this->a;
		temp.b = p.b + this->b;
		return temp;
	}

	int a;
	int b;
};

void test1()
{
	Person p1(10, 10);
	Person p2(20, 20);
	//由p1调用operator+函数，将p2作为参数传递进去
	Person p3 = p1 + p2;
	cout << "p3.a = " << p3.a << endl;
	cout << "p3.b = " << p3.b << endl;
}

//类外的乘号重载
array<int, 5> operator* (const array<int, 5>& arr, int factor)
{
	array<int, 5> temp = arr;
	for (int i = 0; i < temp.size(); i++)
	{
		temp[i] *= factor;
	}
	return temp;
}

void test2()
{
	array<int, 5> arr1 = { 1, 2, 3, 4, 5 };
	//arr1和10只有满足指定的类型，才能调用operator*函数
	arr1 = arr1 * 10;
	for (int i = 0; i < arr1.size(); i++)
	{
		cout << arr1[i] << endl;
	}
}

int main()
{
	test1();
	test2();
	return 0;
}
```

- 算术运算符既可以定义在类内，也可以定义在类外

#### 2.左移运算符重载

- 左移运算符不能在类内重载，因为cout << p中p在<<的右边，而在类内重载时，p调用<<一定是p << ，p在 <<的左边，所以无法实现

##### 最初的重载

```c++
# include <iostream>
using namespace std;

class Person
{
public:
	int age = 18;

};

//cout的类型是ostream(输出流)，所以第一个参数是ostream，第二个参数是Person
void operator<< (ostream& out, const Person& p)
{
	out << "age = " << p.age;
}

int main()
{
	Person p;
	cout << p;
	return 0;
}
```

- [x] 我们顺利地输出了age = 18，但是是否<u>缺了一些什么东西</u>？

​	cout << p的后面没有换行。并不是不想加，而是没法加。因为cout << p的返回值是void，而void是没有办法再调用<<的。

##### 链式调用

```c++
# include <iostream>
using namespace std;

class Person
{
public:
	int age = 18;

};

ostream& operator<< (ostream& out, const Person& p)
{
	out << "age = " << p.age ;
	return out;
}

int main()
{
	Person p;
	cout << p << "666" << endl;
	return 0;
}
```

想要让cout << p继续调用<<，需要让cout << p的返回值是ostream类型

- [x] 既然左移运算符只能定义在类外，如果类内的成员变量是private该怎么办？

##### 全局函数做友元

```c++
# include <iostream>
using namespace std;

class Person
{
	friend ostream& operator<< (ostream& out, const Person& p);
private:
	int age = 18;
};

ostream& operator<< (ostream& out, const Person& p)
{
	out << "age = " << p.age ;
	return out;
}

int main()
{
	Person p;
	cout << p << "666" << endl;
	return 0;
}
```

把该全局重载做友元，就可以让该重载对私有成员生效

至此，我们实现了：

1. 想要重载<<来让他输出类内的内容
2. 发现<<不能重载在类内
3. 重载在类外又不能访问private
4. 把重载函数做友元

#### 3.递增运算符重载

首先，递增运算符是分左右的，并且只有前置递增运算符可以链式调用

```c++
# include <iostream>
using namespace std;

int main()
{
	int a = 10;
	cout << ++a << endl; //11
    //前置递增的返回值是a的引用，可以链式调用
    a = ++(++a);

	int b = 20;
	cout << b++ << endl;  //20
	cout << b << endl;    //21
    //b = (a++)++      //后置递增的返回值是a的副本，是一个临时变量，在a++结束之后就会被销毁，所以不能链式调用

	return 0;
}
```

左递增和右递增必然没法混用，所以需要分别重载

##### 前置递增运算符重载

```c++
# include <iostream>
using namespace std;

class Person
{
	friend ostream& operator<< (ostream& out, const Person& p);
private:
	int age = 18;

public:
	//++(++p)，如果返回值是值而不是引用，++p就无法再将原本的p中的age加一，所以需要返回引用
	Person& operator++()
	{
		//左递增是先加再返回
		age++;
		return *this;
	}
};

ostream& operator<< (ostream& out, const Person& p)
{
	out << "age = " << p.age;
	return out;
}

int main()
{
	Person p;
	cout << ++p << endl;

	return 0;
}
```

- this是指向p的指针常量，*this就是p本身
- 为了实现链式编程，重载函数的返回值必须是引用

##### 后置递增运算符重载

```c++
# include <iostream>
using namespace std;

class Person
{
	friend ostream& operator<< (ostream& out, const Person& p);
private:
	int age = 18;

public:
	//++(++p)，如果返回值是值而不是引用，++p就无法再将原本的p中的age加一，所以需要返回引用
	Person& operator++()
	{
		//左递增是先加再返回
		age++;
		return *this;
	}

	Person operator++(int)
	{
		//右递增是先返回再加
		Person temp = *this;
		age++;
		return temp;
	}
};

ostream& operator<< (ostream& out, const Person& p)
{
	out << "age = " << p.age;
	return out;
}

int main()
{
	Person p;
	cout << ++p << endl;
	cout << ++(++p) << endl;

	Person p2;
	cout << p2++ << endl;
    cout << p2 << endl;

	return 0;
}
```

后置递增运算符返回的是p2的副本，在p2++结束后就会被销毁，无法链式调用

#### 4.赋值运算符重载

首先，我们需要再次回顾以下类在定义时，编辑器会默认提供的东西：

1. 默认构造函数（无参，函数体为空）
2. 默认析构函数（无参，函数体为空）
3. 默认拷贝构造函数，把所有的属性进行值拷贝
4. ==赋值运算符重载operator=，把所有的属性值进行拷贝==

```c++
# include <iostream>
using namespace std;

class Person
{
public:
	int age;

	Person(int age)
	{
		this->age = age;
	}
};

int main()
{
	Person p1(18);
    //编辑器默认对=进行了重载，可以把一个对象的值全部赋给另一个对象
	Person p2 = p1;
	cout << p2.age << endl;

	return 0;
}
```

------

我们需要再次回顾一下**深拷贝**与**浅拷贝**：

- 直接把一块地方挪到另一块地方叫浅拷贝
- 重新开一块地方，把原来的东西复制一份叫深拷贝

- [x] 对于**纯值类型**的变量，如int ,float，**只存在深拷贝**

```c++
int a = 10;
int b = a;
```

很明显，b重新开了一块地方，把a复制了过去，所以是深拷贝

------

在默认情况下，类中的=直接把所有的属性进行值拷贝，对于**纯值类型**变量来说当然是**深拷贝**。但是如果类内有指针，对于指针来说，则做的是**浅拷贝**，因为两个指针的值相同，指向同一块区域

```c++
# include <iostream>
using namespace std;

class Person
{
public:
	int* age;

	Person(int age)
	{
		this->age = new int(age);  //类内的指针age指向堆区的一块内存
	}
};

int main()
{
	Person p1(18);
    //类中的指针age发生了浅拷贝，p1.age和p2.age指向同一块区域
	Person p2 = p1;
	cout << *p2.age << endl;    //18

	return 0;
}
```

- [x] 看起来好像没啥问题，p2也可以正常访问age的值，那么这样到底哪里有问题？
  - 众所周知，new会在堆区开一块内存，而这块内存**不会主动释放**，所以我们需要用析构函数来释放这块内存
  - 如果使用浅拷贝，会导致p1.age和p2.age指向同一块内存，而二者的析构函数都会释放这块内存，导致报错

为了避免报错，我们需要手动实现深拷贝

```c++
# include <iostream>
using namespace std;

class Person
{
public:
	int* age;
	Person(){}
	Person(int age)
	{
		this->age = new int(age);  //类内的指针age指向堆区的一块内存
	}

	~Person()
	{
		if (*age != NULL)
		{
			//如果age指向的地方不为空，则释放age指向的内存，并让age变成空指针
			delete age;
			age = NULL;
		}
	}

	//手动实现深拷贝
	void operator= (Person& p)
	{
		//如果this的age初始化过（即已经指向了一块内存），那么需要先把这块内存释放掉
		if (age != NULL)
		{
			delete age;
		}
		//重新开一块新区域，值为p的age指向的值
		age = new int(*p.age);
	}
};

int main()
{
	Person p1(18);
	Person p2 = p1;
	cout << *p2.age << endl;    //18
	
	return 0;
}
```

通过这种方式，我们让p1和p2的age指向两个地址不同，但是值相同的区域

实现链式调用的方法：

```c++
# include <iostream>
using namespace std;

class Person
{
public:
	int* age;
	Person(){}
	Person(int age)
	{
		this->age = new int(age);  //类内的指针age指向堆区的一块内存
	}

	~Person()
	{
		if (*age != NULL)
		{
			//如果age指向的地方不为空，则释放age指向的内存，并让age变成空指针
			delete age;
			age = NULL;
		}
	}

	//手动实现深拷贝
	Person& operator= (Person& p)
	{
		//如果this的age初始化过（即已经指向了一块内存），那么需要先把这块内存释放掉
		if (age != NULL)
		{
			delete age;
		}
		//重新开一块新区域，值为p的age指向的值
		age = new int(*p.age);
		//将this返回，就可以实现链式调用
		return *this;
	}
};

int main()
{
	Person p1(18);
	Person p2(20);
	Person p3(22);
	p3 = p2 = p1;
	cout << *p3.age << *p2.age << *p1.age << endl;    //都会变成18
	
	return 0;
}
```

#### 5.关系运算符重载

关系运算符有很多，这里以==为例

```c++
int a = 10;
int b = 10;
cout << (a == b) << endl;   //输出1
```

我们要实现的就是让==能够判断两个对象是否相等

```c++
# include<iostream>
using namespace std;

class Person
{
public:
	int age;
	Person(int age)
	{
		this->age = age;
	}

	bool operator== (Person& p)
	{
		if (this->age == p.age)
		{
			return true;
		}
		else
		{
			return false;
		}
	}
};

void test1(Person& p1, Person& p2)
{
	if (p1 == p2)
	{
		cout << "两者是相等的" << endl;
	}
	else
	{
		cout << "两者是不等的" << endl;
	}
}

int main()
{
	Person p1(18);
	Person p2(20);
	test1(p1, p2);

	return 0;
}
```

比较简单，没啥可说

#### 6.函数调用运算符重载（仿函数）

函数调用运算符就是()，由于重载后的使用方式非常像函数的调用，所以被称为==仿函数==

- [x] 首先，先看一下一般情况下，用一个函数print_age来打印类中的age

```c++
# include <iostream>
using namespace std;

class Person
{
public:
	int age;
	Person(int age)
	{
		this->age = age;
	}
};

//函数定义
void print_age(Person& p)
{
	cout << p.age << endl;
}

int main()
{
	Person p1(18);
    //函数调用
	print_age(p1);
	return 0;
}
```

- [x] 然后是仿函数

```c++
# include <iostream>
using namespace std;

class Person
{
public:
	int age;
	Person(int age)
	{
		this->age = age;
	}

	void operator() ()
	{
		cout << age << endl;
	}
};

int main()
{
	Person p1(18);
	p1();
	return 0;
}
```
