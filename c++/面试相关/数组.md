```c++
char a[] = "12345";

char* p = a;

cout << sizeof(a) << endl;  //6
cout << sizeof(p) << endl;  //8

cout << strlen(a) << endl;  //5
cout << strlen(p) << endl;  //5

const char* q = "World";
cout << sizeof(q) << endl;  //8
cout << strlen(q) << endl;  //5
```

1. 字符数组a在初始化时，会在12345后面加一个**空字符'\0'**来表示这是字符串的末尾，所以sizeof会输出6
2. 数组名a在很多时候会**退化**为指向第一个元素的指针，但是他的真实含义**绝对不是指向第一个元素的指针**，而是包含**整个数组**的一块整体；而指针p真的就**只是一个指针**，他并**不知道**自己指向的是一块连续的数组内存，所以sizeof就是p的大小8
3. **strlen**是一个专用于数组的函数，他知道字符数组的结尾是'\0'，所以他不会把'\0'算进去，所以输出5
4. **strlen**会以指针的形式取寻找'\0'，所以他会沿着p指针寻找，最后输出也是5
5. 字符串字面量返回的是他首个元素的地址。q只是一个指针，所以sizeof自然是8
6. strlen会沿着指针q去寻找'\0'，所以是5
