### 指向普通对象的指针

#### 1.指向非结构体对象的指针

```c++
int a = 1;
int* p = &a;//初始化指针
*p = 2;//调用指针
```

#### 2.指向结构体对象的指针

```c++
class Person {
public:
	int a = 0;
	void print(int num) {
		std::cout << num << std::endl;
	}
};

Person p;
Person* ptr = &p; //初始化

cout << ptr->a << endl; //调用成员变量
ptr->print(2);       //调用成员函数
```

### 指向常量对象的指针：常量指针

#### 1.指向非结构体对象的常量指针

> [!WARNING]
>
> 常量指针并不一定指向常量，常量指针只是无法修改指向的对象

```c++
int a = 1;
const int* p = &a; //初始化

cout << *p << endl; //调用指针，由于是const指针，所以不能修改对象a
```

#### 2.指向结构体对象的常量指针

> [!WARNING]
>
> 常量指针ptr的this指针是const Person * const型，他可以指向int a，但是不能调用void print(int num)，因为print函数需要的this指针是Person* const型，无法接收const Person* const型

```c++
class Person {
public:
	int a = 0;
	void print  (int num) const {
		std::cout << num << std::endl;
	}
};

Person p;

const Person* ptr = &p; //初始化

cout << ptr->a << endl; //调用成员变量

ptr->print(2);//调用const成员函数
```

### 指向函数的指针：函数指针

> [!WARNING]
>
> 函数指针声明时需要确定函数的**返回值**和**参数列表**
>
> 函数指针调用的时候可以加*，但是不加也可以，编译器会**自动解引用**

```c++
void print(int num) {
	std::cout << num << std::endl;
}

void (*p)(int); //函数指针声明
p = &print;   //函数指针初始化

p(2);  //函数指针调用
```

### 指向对象成员的指针

#### 1.指向成员变量的指针

> [!WARNING]
>
> 指向成员变量的指针首先是一个指向变量的指针，int* ptr;
>
> 然后这个变量是属于对象的，ptr = &p.a;

```c++
Person p;

int* ptr; //指针声明
ptr = &p.a;  //指针初始化 
cout << *ptr << endl; //调用指针
```

#### 2.指向成员函数的指针

> [!WARNING]
>
> 成员函数的地址并不是内存中的地址，如果函数是非虚函数，它存储的是该函数在类方法区中的**偏移量**；如果不指定指针在类作用域内，指针完全无法找到这个偏移量

```c++
Person p;

void (Person:: * ptr)(int);  //指针声明

ptr = &Person::print;  //指针初始化

(p.*ptr)(2);  //调用指针
```
