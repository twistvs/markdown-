- **左值**是表达式结束后依旧存在的**持久对象**，**右值**是表达式结束后就不再存在的**临时对象**

- **左值引用**就是我们常说的**别名**，必须绑定在一个左值上
- **右值引用**绑定右值，但是右值引用本质上是一个左值，因为他在表达式结束后依旧存在

```c++
# include <iostream>

int main() {
	int a = 10;
	//左值引用必须绑定左值
	int& b = a;
    //int & b = 10  错误，10作为字面量，是右值
    
    //右值引用绑定右值
	int&& c = 10;

	std::cout << c << std::endl;
	
	return 0;
}
```
