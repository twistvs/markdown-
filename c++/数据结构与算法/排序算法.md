#### 冒泡排序

- 将数组从小到大排列，通过**迭代比较相邻两数**，把大的放到后面，每一轮把最大的放到最后面，总共n-1轮
- O(n^2^)   O(1)   稳定
- 需要i和j两个计数，i和j始终是相邻关系

```c++
void mySort::BubbleSort(std::vector<int>& vec)
{
	for (int j = vec.size() - 1; j >= 1; j--)
	{
		for (int i = 0; i < j; i++)
		{
			if (vec[i] > vec[i + 1])
			{
				std::swap(vec[i], vec[i + 1]);
			}
		}
	}
}
```

如果在某一趟排序之后，数组已经有序，则后面的每趟排序都是在浪费时间

```c++
void mySort::BubbleSort(std::vector<int>& vec)
{
	for (int j = vec.size() - 1; j >= 1; j--)
	{
        //默认数组已经有序
		bool flag = true;
		for (int i = 0; i < j; i++)
		{
			if (vec[i] > vec[i + 1])
			{
				std::swap(vec[i], vec[i + 1]);
                //只要这一趟排序发生了一次交换，就使flag为false
				flag = false;
			}
		}
        //如果这一趟排序结束之后，flag还是true，说明这一趟没有发生交换，即数组已经有序，直接break
		if (flag == true)
		{
			break;
		}
	}
}
```

#### 选择排序

- 选择排序是**冒泡排序的改进**

- 通过添加第三个计数minIndex来存储每一轮中最小的值，来减少每一轮的交换次数
- i始终是该轮的第一个值，minIndex指向最小的值，j从i开始向右遍历，每一轮结束后，将i和minIndex互换，并重置i，j，minIndex的位置

- O(n^2^)  O(1)   不稳定（如果有*20*，20，5。经过一轮后，第一个*20*会被换到第二个20的右边）

```c++
void mySort::SelectSort(std::vector<int>& vec)
{
	int minIndex = 0;
	for (int i = 0; i < vec.size() - 1; i++)
	{
		for (int j = i + 1; j < vec.size(); j++)
		{
			if (vec[j] < vec[minIndex])
			{
				minIndex = j;
			}
		}
		std::swap(vec[i], vec[minIndex]);
		minIndex = i + 1;
	}
}
```

#### 直接插入排序

- 将数组分为左有序区，右无序区，将无序区的第一个元素逐个插入
- O(n^2^)，O(1)，稳定

```c++
void mySort::InsertSort(std::vector<int>& vec)
{
	for (int i = 1; i < vec.size(); i++)
	{
		for (int j = 0; j < i; j++)
		{
			if (vec[i] < vec[j])
			{
				int temp = vec[i];
				for (int k = i; k > j; k--)
				{
					vec[k] = vec[k - 1];
				}
				vec[j] = temp;
				//当i成功插入到有序区里面，就不再需要继续遍历有序区，直接跳到下一次循环
				break;
			}
		}
	}
}
```

只要无序区的第一个元素插到了有序区里面，就立刻跳出有序区的遍历

#### 希尔排序（缩小增量排序）

- 希尔排序是**直接插入排序的改进**
- O(n^1.3^)，O(1)，不稳定

1. 计算初始增量gap = n/2，根据增量把数列分成很多组，分别进行插入排序
2. 将gap再除以2，再分别插入排序
3. 直到gap等于1，就是一趟普通的插入排序

#### 堆排序

- O(nlogn)，O(1)，不稳定

堆排序每一轮都构建一个大根堆，然后把最大的root弹出来，再把最后一个元素提到root上

堆的第一次构建需要自底向上，后续的每一次构建都只需要对root进行siftDown

```c++
//利用大根堆实现堆排序
# include <iostream>
# include <vector>

//将下标为start的子树进行堆排序，会一直循环到树的最底部
//siftDown只能把start《下沉》到他该待的位置，而不能保证下面的子树也都能待在他们该待的位置，所以要从底部开始一个个下沉
/*从性质上来讲，如果start的左右子树已经是大顶堆了，那么siftDown就可以直接一口气把start变成大顶堆，所以在调换完顺序后，root的
左右子树都是大顶堆，所以可以直接用一次siftDown构建大顶堆*/
void siftDown(std::vector<int>& vec, int start, int n) {
	int i = start;
	int left = i * 2 + 1;
	int right = i * 2 + 2;
	int largest = i;
	while (true) {
		//三个元素比大小并不一定非要放在一块比，先比前两个，再比第三个，类似于冒泡排序的思想，一样可以比出最大值
		if (left < n && vec[left] > vec[largest]) {
			largest = left;
		}
		if (right < n && vec[right] > vec[largest]) {
			largest = right;
		}
		if(largest != i) {
			std::swap(vec[i], vec[largest]);
			i = largest;
			left = i * 2 + 1;
			right = i * 2 + 2;
		}
		//如果largest == i，说明已经是大顶堆了，循环结束
		else {
			break;
		}
	}
}

void heapSort(std::vector<int>& vec) {
	int n = vec.size();
	//从第一个非叶子的节点开始构建大顶堆，对于完全二叉树，第一个非叶子节点的下标为n/2-1
	for (int i = n / 2 - 1; i >= 0; i--) {
		siftDown(vec, i, n);
	}

	//构建完大顶堆，开始进行排序，每一次把堆顶（即最大的元素放在vec的最后面)，然后再次把堆调成大顶堆
	//由于root的左右子树都是大顶堆，所以只需要一次siftDown就可以
	for (int i = n - 1; i >= 0; i--) {
		std::swap(vec[0], vec[i]);
		siftDown(vec, 0, i);
	}
}

int main() {
	std::vector<int> vec({ 2, 3, 2, 4, 1, 6, 7, 3 });
	heapSort(vec);
	for (auto i : vec) {
		std::cout << i;
	}
	return 0;
}
```

#### 归并排序

- 归并排序是基于**递归**的排序
- 归并排序递归的把数列**两等分**，分到只剩1个或0个元素的时候进行合并merge
- O(nlogn)，O(1)，稳定

```c++
//对链表的归并排序
class Solution {
public:
    ListNode* sortList(ListNode* head) {
        //递归结束条件，排序的数组的元素小于等于1
        if(head == NULL || head->next == NULL){
            return head;
        }

        //快慢指针找链表中点
        ListNode* slow = head;
        ListNode* fast = head;

        while(fast->next && fast->next->next){
            slow = slow->next;
            fast = fast->next->next;
        }

        ListNode* mid = slow->next;
        //不要忘了把前半部分的尾巴断掉
        slow->next = NULL;

        ListNode* left = sortList(head);
        ListNode* right = sortList(mid);

        return merge(left, right);
    }

    ListNode* merge(ListNode* left, ListNode* right){
        ListNode* dummy = new ListNode(0);
        ListNode* temp = dummy;

        while(left && right){
            if(left->val < right->val){
                temp->next = left;
                temp = temp->next;
                left = left->next;
            }
            else{
                temp->next = right;
                temp = temp->next;
                right = right->next;
            }
        }

        if(left){
            temp->next = left;
        }
        if(right){
            temp->next = right;
        }
        return dummy->next;
    }
};
```

#### 快速排序

- O(nlogn)，O(nlogn)，不稳定

```c++
void quickSort(vector<int>& nums, int left, int right) {
    if (left >= right) return;

    int pivot = nums[left]; // 简单起见，取最左侧为基准
    int i = left, j = right;

    while (i < j) {
        // 从右向左找第一个小于 pivot 的数
        while (i < j && nums[j] >= pivot) j--;
        // 从左向右找第一个大于 pivot 的数
        while (i < j && nums[i] <= pivot) i++;
        
        if (i < j) swap(nums[i], nums[j]);
    }
    // 将基准值归位
    swap(nums[left], nums[i]);

    // 递归处理左右两部分
    quickSort(nums, left, i - 1);
    quickSort(nums, i + 1, right);
}
```
