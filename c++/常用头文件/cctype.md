| **函数**         | **作用**                                       | **示例**                       |
| ---------------- | ---------------------------------------------- | ------------------------------ |
| **`isalpha(c)`** | 判断是否为字母                                 | `'A'` -> true, `'1'` -> false  |
| **`isdigit(c)`** | 判断是否为数字                                 | `'1'` -> true, `'A'` -> false  |
| **`islower(c)`** | 判断是否为小写字母                             | `'a'` -> true, `'A'` -> false  |
| **`isupper(c)`** | 判断是否为大写字母                             | `'A'` -> true, `'a'` -> false  |
| **`isspace(c)`** | 判断是否为空白字符（空格、换行、制表符等）     | `' '` -> true, `'\n'` -> true  |
| **`tolower(c)`** | 将大写字母转为小写（本身是小写或非字母则不变） | `'A'` -> `'a'`, `'1'` -> `'1'` |
| **`toupper(c)`** | 将小写字母转为大写                             | `'a'` -> `'A'`                 |
