### 1.遍历算法

#### for_each

`for_each`  （起始迭代器，结束迭代器，对该元素的操作）   包含在algorithm中

- [x] 首先看看自己遍历vector可以怎么写

```c++
#include <algorithm>
#include <vector>

int main()
{
	std::vector<int> a;
	for (int i = 1; i < 6; i++)
	{
		a.push_back(i);
	}
    //利用迭代器，可以遍历vector
	for (auto it = a.begin(); it != a.end(); it++)
	{
		std::cout << *it << std::endl;
	}
	
	return 0;
}
```

- [x] 利用for_each函数。先看普通函数作为参数

```c++
#include <iostream>
#include <algorithm>
#include <vector>

void print(int a)
{
	std::cout << a << std::endl;
}

int main()
{
	std::vector<int> a;
	for (int i = 1; i < 6; i++)
	{
		a.push_back(i);
	}
    //行为函数既可以是函数指针，也可以是函数对象
	std::for_each(a.begin(), a.end(), print);

	return 0;
}
```

再看仿函数作为参数

```c++
#include <iostream>
#include <algorithm>
#include <vector>

class print
{
public:
	void operator() (int a)
	{
		std::cout << a << std::endl;
	}

    //依旧匿名对象

}
```

#### transform

`transform(原容器开始迭代器，原容器结束迭代器，目标容器开始迭代器，对元素的操作)`

把原容器中指定范围的内容经过一些操作后复制到目标容器中

```c++
#include <iostream>
#include <vector>
#include <algorithm>

//双倍
class Double
{
public:
	int operator() (int a)
	{
		int b = a * 2;
		return b;
	}
};

//打印
class Print
{
public:
	void operator() (int a)
	{
		std::cout << a << std::endl;
	}
};

int main()
{
	std::vector<int> a;
	for (int i = 1; i < 6; i++)
	{
		a.push_back(i);
	}
	std::vector<int> b;
	//目标容器需要预先开辟空间，否则挪不进去
	b.resize(a.size());

	//将a中的内容双倍后给b
	std::transform(a.begin(), a.end(), b.begin(), Double());
	std::for_each(a.begin(), a.end(), Print());     //1，2，3，4，5
	std::for_each(b.begin(), b.end(), Print());     //2，4，6，8，10

	return 0;
}
```

### 2.查找算法

#### find

`find(开始迭代器，结束迭代器，查找内容)`

查找内置数据类型（如int）

```c++
#include <iostream>
#include <algorithm>
#include <vector>

//查找内置数据类型（如int）
void test1()
{
	std::vector<int> a;
	for (int i = 1; i < 6; i++)
	{
		a.push_back(i);
	}
	std::vector<int>::iterator it = std::find(a.begin(), a.end(), 7);
	if (it == a.end())
	{
		std::cout << "没找到" << std::endl;
	}
	else
	{
		std::cout << "找到了" << std::endl;
	}
}


int main()
{
	test1();

	return 0;
}
```

------

如果查找的是自定义的数据类型，则可能出现以下问题：

```c++
class Person
{
public:
	int age;
	std::string name;
	Person(int age, std::string name)
	{
		this->age = age;
		this->name = name;
	}
};

void test2()
{
	Person p1(18, "aaa");
	Person p2(20, "bbb");
	Person p3(18, "ccc");
	std::vector<Person> a;
	a.push_back(p1);
	a.push_back(p2);
	a.push_back(p3);
	//要在vector中找p2
	std::vector<Person>::iterator it = std::find(a.begin(), a.end(), p2);

}

int main()
{
	test2();

	return 0;
}
```

在find函数内，会把元素与p2做比较，也就是if(p ==  p2)。但是类内并不自带==的重载，find没办法判断p是否等于p2

为了实现两个对象的相等判断，我们需要在类内重载==

```c++
#include <iostream>
#include <algorithm>
#include <vector>
#include <string>

class Person
{
public:
    //类内重载==
	bool operator== (const Person& p)
	{
		if (this->age == p.age && this->name == p.name)
		{
			return true;
		}
		else
		{
			return false;
		}
	}
	
	Person(int age, std::string name)
	{
		this->age = age;
		this->name = name;
	}

	int age;
	std::string name;
};

void test2()
{
	Person p1(18, "aaa");
	Person p2(20, "bbb");
	Person p3(18, "ccc");
	std::vector<Person> a;
	a.push_back(p1);
	a.push_back(p2);
	a.push_back(p3);
	//要在vector中找p2
	std::vector<Person>::iterator it = std::find(a.begin(), a.end(), p2);
	if (it == a.end())
	{
		std::cout << "没找到" << std::endl;
	}
	else
	{
		std::cout << "找到了 " << "age为" << it->age << "name为" << it->name << std::endl;
	}
}

int main()
{
	test2();

	return 0;
}
```

#### find_if

`find_if(开始迭代器，结束迭代器，谓词)`

- find找的是==，而find_if找的是一个自定义的条件

依旧先查找内置数据类型

```c++
#include <iostream>
#include <algorithm>
#include <vector>

class Greater5
{
public:
	bool operator() (int a)
	{
		if (a > 5)
		{
			return true;
		}
		else
		{
			return false;
		}
	}
};

//查找内置数据类型
void test1()
{
	std::vector<int> a;
	for (int i = 0; i < 11; i++)
	{
		a.push_back(i);
	}

    //find_if和find还是很相似的
	std::vector<int>::iterator it = std::find_if(a.begin(), a.end(), Greater5());
	if (it == a.end())
	{
		std::cout << "没找到" << std::endl;
	}
	else
	{
		std::cout << "找到了" << std::endl;
	}
}

int main()
{
	test1();
	return 0;
}
```

------

然后是查找自定义数据类型

```c++
#include <iostream>
#include <algorithm>
#include <vector>

class Person
{
public:
	Person(int a, int h):age(a), height(h) {}
	
	int age;
	int height;
};

class BigPerson
{
public:
	bool operator() (const Person& p)
	{
		if (p.age > 18 && p.height > 180)
		{
			return true;
		}
		else
		{
			return false;
		}
	}
};

//查找自定义数据类型
void test2()
{
	std::vector<Person> a;
	Person p1(15, 155);
	Person p2(20, 190);
	Person p3(19, 188);
	a.push_back(p1);
	a.push_back(p2);
	a.push_back(p3);

	std::vector<Person>::iterator it = std::find_if(a.begin(), a.end(), BigPerson());
	if (it == a.end())
	{
		std::cout << "没找到" << std::endl;
	}
	else
	{
		std::cout << "age为" << it->age << ",height为" << it->height << std::endl;
	}
}

int main()
{
	test2();
	return 0;
}
```

由于find_if中的谓词必须要自己写，所以反而避免了find中忘记重载==的情况

#### adjacent_find

`adjacent_find(开始迭代器，结束迭代器)`

查找容器中第一个相邻重复的部分（如1,1），返回第一个元素的迭代器

```c++
#include <iostream>
#include <algorithm>
#include <string>

void test1()
{
	std::string s;
	s = "abacbnmqiqbbccc";
	std::string::iterator it = std::adjacent_find(s.begin(), s.end());
	if (it == s.end())
	{
		std::cout << "没找到" << std::endl;
	}
	else
	{
		std::cout << "重复的是" << *it << std::endl;
	}
}

int main()
{
	test1();
	return 0;
}
```

#### binary_search

`binary_search(开始迭代器，结束迭代器，查找内容)`  ，返回值为bool，仅仅只说明有没有找到，而不会返回元素的位置

- [x] 明明find可以把位置找出来，为啥还要弄一个不能找的？
  ——因为这个查找用的是二分查找，速度更快

```c++
#include <iostream>
#include <vector>
#include <algorithm>

void test1()
{
	std::vector<int> a;
	a.push_back(1);
	a.push_back(2);
	a.push_back(3);

	if (std::binary_search(a.begin(), a.end(), 3))
	{
		std::cout << "有3" << std::endl;
	}
	else
	{
		std::cout << "没3" << std::endl;
	}
}

int main()
{
	test1();
	return 0;
}
```

### 3.排序算法

#### sort

- sort是`<algorithm>`头文件

```c++
//默认是从小到大排序
std::sort(v.begin(), v.end());

//从大到小排序的方法
//使用greater
std::sort(v.begin(), v.end(), greater<int>());

//使用lambda表达式
std::sort(v.begin(), v.end(), [](int x, int y) {
	return x > y;
});

//使用反向迭代器，这两个迭代器是专门为反向遍历而设计的
std::sort(v.rbegin(), v.rend());
```

#### next_permutation

- next_permutation既会直接改变容器的内容，也有返回值，当容器拥有下一个排列时，返回true,如果已经时最后一个排列，则返回false

```c++
std::vector<int> v = {1, 2, 3};

std::next_permutation(v.begin(), v.end());

printVec(v);//1， 3 ，2
```
