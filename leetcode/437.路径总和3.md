#### 双重嵌套dfs

1. 用外层的dfs遍历所有的节点‘
2. 用内层的dfs从当前节点开始，向下dfs并得到所有满足的路径
3. 每条路径都满足从当前节点开始

```c++
class Solution {
public:
    int rootSum(TreeNode* root, long long targetSum){
        if(root == NULL){
            return 0;
        }

        int ret = 0;
        if(root->val == targetSum){
            ret++;
        }

        ret = ret + rootSum(root->left, targetSum - root->val);
        ret = ret + rootSum(root->right, targetSum - root->val);
        return ret;
    }

    int pathSum(TreeNode* root, int targetSum) {
        if(root == NULL){
            return 0;
        }

        int ret = rootSum(root, targetSum);
        ret = ret + pathSum(root->left, targetSum);
        ret = ret + pathSum(root->right, targetSum);
        return ret;
    }
};
```

#### dfs+前缀和

- 在深搜的过程中，我们往下走的路径，会形成一个前缀和，比如10, 5, 3, 3.随着这条路往下走，会得到10，15，18，21的前缀和
- 只要当前的总路径和减去前面的某个前缀和等于targetsum，那么就存在一条以当前节点为尾的路径和等于targetsum

1. ------

   使用一个map来记录某个前缀和出现的次数

2. 每当访问一个新节点，更新前缀和map，并判断是否有满足条件的前缀和

3. 在向上返回的过程中，前缀和数组map和路径和currSum也要跟着回溯

```c++
class Solution {
public:
    //long是前缀和，int是当前路径存在几个等于currSum-targetSum的前缀和
    unordered_map<long, int> prefix;    

    //返回值代表对于节点root，包括他自己和他下面的所有节点，一共有多少条路径等于targetSum
    int dfs(TreeNode* root, long currSum, int targetSum){
        if(root == NULL){
            return 0;
        }

        currSum += root->val;
        //对于根节点root来讲，如果他的值等于targetsum，则ans就会等于prefix[0]，即等于1，他自己就是一条路径
        int ans = prefix[currSum - targetSum];
        
        //currSum是以值传递的形式参与递归，不需要回溯；prefix是外部的变量，需要回溯
        prefix[currSum]++;
        ans += dfs(root->left, currSum, targetSum);
        ans += dfs(root->right, currSum, targetSum);
        prefix[currSum]--;

        return ans;
    }

    int pathSum(TreeNode* root, int targetSum) {
        //不管是什么路径，都肯定至少存在一个等于0的前缀和，即什么节点都没有的路径，所以把prefix[0]初始化为1（但是后面可能还会变大，因为可能存在2,-2这样的路径）
        prefix[0] = 1;
        return dfs(root, 0, targetSum);
    }
};
```
