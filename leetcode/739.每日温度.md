#### 单调栈

- 维护一个单调栈index，里面存的是元素序号，并且温度保持从大到小排列
- 如果新遍历的元素温度比top高，那么进入循环，把index中所有比该元素小的元素都pop()出来，并且计算这些被pop出来的天数的ans值（ans值就是用当前的天数减去pop出来的元素的天数）

```c++
class Solution {
public:
    vector<int> dailyTemperatures(vector<int>& temperatures) {
        int n = temperatures.size();
        vector<int> ans(n, 0);

        stack<int> index;

        index.push(0);

        for(int i = 1; i < n; i++){
            //如果第i天的温度比top高，那么循环遍历栈，把栈中所有比第i天温度低的天数pop出，并且计算他们的ans
            //判断!index.empty()是因为可能所有栈内元素都比第i天小，这样会把栈直接清空，当栈空了之后，循环也会结束
            while(!index.empty() && temperatures[i] > temperatures[index.top()]){
                ans[index.top()] = i - index.top();
                index.pop();
            }
            //如果第i天的温度比top低，那么index就还是单调的，直接把i插入到栈里面
            index.push(i);
        }

        return ans;
    }
};
```
