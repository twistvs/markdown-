#### 递归

- 先将左子树反转，再将右子树反转，然后把左儿子和右儿子交换

```c++
class Solution {
public:
    TreeNode* invertTree(TreeNode* root) {
        if(root == NULL){
            return NULL;
        }
		
        //对于叶子节点，他的左儿子和右儿子都是NULL
        TreeNode* left = invertTree(root->left);
        TreeNode* right = invertTree(root->right);

        root->left = right;
        root->right = left;

        return root;
    }
};
```
