#### 前缀和+哈希表

![image-20260324154502226](C:\Users\lee\AppData\Roaming\Typora\typora-user-images\image-20260324154502226.png)

- 这道题的递归是递归子数组的右边界，对于某个右边界，我们要判断两种情况:

1. 从0到右边界的子数组和是否为k？即前缀和pre是否为k？
2. 从除了0之外的其他左边界到这个右边界的子数组和是否为k？这个问题可以转化为从0到左边界是否为pre-k，即前缀和pre是否为pre-k？
3. 只要判断这两种情况，就可以获得全部和为k的子数组

------

- 如果我们搞一个双循环，判断每个右边界i的所有左边界j与0的子数组是否为pre-k，就有点麻烦了；由于在遍历的过程中，本来就会判断所有的pre，所以可以把pre的每个值出现的次数记录下来，有几个pre-k出现，就说明有几个和为k的子数组

```c++
class Solution {
public:
    int subarraySum(vector<int>& nums, int k) {
        //记录前缀和
        unordered_map<int, int> mp;
        int pre = 0;
        int ans = 0;

        for(int num : nums){
            pre += num;
            if(pre == k){
                ans++;
            }
            if(mp.count(pre - k) != 0){
                ans += mp[pre - k];
            }
            mp[pre]++;
        }

        return ans;
    }
};
```

------

- 如果不把0到i单独拿出来判断，我们可以设初始值mp[0] = 1，在初始状态下，前缀和为0的子数组只有空数组这一个，这样就可以去掉if(pre == k)的判断

```c++
class Solution {
public:
    int subarraySum(vector<int>& nums, int k) {
        //记录前缀和
        unordered_map<int, int> mp;
        int pre = 0;
        int ans = 0;

        //初始化
        mp[0] = 1;

        for(int num : nums){
            pre += num;
            if(mp.count(pre - k) != 0){
                ans += mp[pre - k];
            }
            mp[pre]++;
        }

        return ans;
    }
};
```

#### 任意起点的路径和

- 这道题其实可以看做从任意起点开始，往右走的某一条路径和为k的个数，和那道二叉树路径和其实是同理的

**前缀和与路径和的天然相性**

- 通过记录sum和前缀和，我们可以通过遍历路径结束点的方式来找到所有满足条件的路径
