#### 环型贪心

- 如果从1走到5的时候总油量变成负了，说明：

1. 从1开始是**不可能走完环**的，毕竟连5都走不到，所以以1作为起点的可能性是不存在的
2. 从2，3，4为起点开始也是不可能走完环的，因为我们从1往5走时，在2，3，4的时候都是正的，在这种情况下都走不到5，从2，3，4开始是0油出发，更不可能走到5了，连5都走不到，更不可能走完环

```c++
class Solution {
public:
    int canCompleteCircuit(vector<int>& gas, vector<int>& cost) {
        int total_gas = 0;
        int total_cost = 0;
        int curr_tank = 0; // 当前路段剩余油量
        int start_index = 0; // 假设的起点

        for (int i = 0; i < gas.size(); i++) {
            total_gas += gas[i];
            total_cost += cost[i];
            curr_tank += gas[i] - cost[i];

            // 如果当前油箱小于0，说明从 start_index 到 i 这一段走不通
            if (curr_tank < 0) {
                // 将起点重置为 i + 1
                start_index = i + 1;
                // 重置当前油箱
                curr_tank = 0;
            }
        }

        // 如果总油量小于总消耗，无解
        return (total_gas < total_cost) ? -1 : start_index;
    }
};
```

- 根据以上推导，我们知道如果最后的start_index为x，则x之前的**所有起点**都是不成立的
- 如果从x+1开始，起始油为0，而从x开始时，走到x+1时油为正，很明显，从x开始更有可能走到终点
- 也就是说(0, x-1)不可能走到终点，而(x, n - 1)中，x最有可能到达终点，所以x就是所有起点中的最优点，如果total_gas >= total_cost，那么x就是那个唯一解
