**前缀树可以实现的功能：**

1. `void insert(string word)` 向前缀树中插入字符串word
2. `bool search(string word)` 在前缀树中查询字符串word，如果存在则返回true
3. `bool startWith(string prefix)` 在前缀树中查询前缀prefix，如果存在则返回true



**注意事项**

1. 前缀树本质上是一棵树，从树根往外发散形成了多条路径
2. 查询操作要求前缀树中存在与word完全匹配的字符串，所以需要用一个标志位isEnd来标明字符串的结尾

**需要达成的目标：**

1. 每个节点都有自己的值`val`
2. 对于每个节点来讲，他都有可能向26个方向发散，所以需要一个`map`来记录这26个方向分别指向哪里
3. 每个节点都需要记录自己是不是一个word的结尾，需要一个标志位`isEnd`



```c++
struct Treenode{
    char val;
    unordered_map<char, Treenode*> next;
    bool isEnd;

    Treenode(char val){
        this->val = val;
        isEnd = false;
    }
};


class Trie {
public:
    //先搞一个头节点
    Treenode* root = new Treenode('!');

    Trie() {
        
    }
    
    //已经有的字符直接把temp往后挪，没有的就插到后面，在整个word遍历完之后，不要忘了给结尾的节点设置isEnd = true
    void insert(string word) {
        Treenode* temp = root;
        for(char s : word){
            if(temp->next.count(s) == 0){
                temp->next[s] = new Treenode(s);
            }
            //不管s是否已经存在，temp都需要后移，所以不需要判断
            temp = temp->next[s];
        }
        temp->isEnd = true;
    }
    
    //如果有的字符没有，直接返回false，如果遍历到最后都有，判断最后一位是不是isEnd，如果是返回true
    bool search(string word) {
        Treenode* temp = root;
        for(char s : word){
            if(temp->next.count(s) == 0){
                return false;
            }
            temp = temp->next[s];
        }
        if(temp->isEnd == true){
            return true;
        }
        else{
            return false;
        }
    }
    
    //相比于search，prefix更简单，只要能够顺利遍历完word，就直接返回true
    bool startsWith(string prefix) {
        Treenode* temp = root;
        for(char s : prefix){
            if(temp->next.count(s) == 0){
                return false;
            }
            temp = temp->next[s];
        }
        return true;
    }
};
```
