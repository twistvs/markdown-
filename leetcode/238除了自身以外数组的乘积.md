很明显，对于元素i，除了他以外的乘积等于他左边所有元素的乘积和他右边所有元素的成绩

1. 构建数组L，L[i]记录了i左边所有元素的乘积
2. 构建数组R，R[i]记录了i右边所有元素的乘积
3. 构建数组answer，answer[i] = L[i] * R[i]，得到结果

L和R都需要遍历一次nums来获得所有值，answer也需要遍历一遍来获得乘积

```c++
class Solution {
public:
    vector<int> productExceptSelf(vector<int>& nums) {
        int n = nums.size();
        vector<int> L(n, 0);
        vector<int> R(n, 0);
        vector<int> answer(n ,0);

        for(int i = 0; i < n; i++){
            if(i == 0){
                L[i] = 1;
            }
            else{
                L[i] = L[i - 1] * nums[i - 1];
            }
        }

        for(int j = n - 1; j >= 0; j--){
            if(j == n - 1){
                R[j] = 1;
            }
            else{
                R[j] = R[j + 1] * nums[j + 1];
            }
        }

        for(int i = 0; i < n; i++){
            answer[i] = L[i] * R[i];
        }
        return answer;
    }
};
```
