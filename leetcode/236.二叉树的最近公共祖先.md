- return只能把信息从儿子节点往父亲节点传，如果信息传递的方向不是从儿子到父亲，就必要要用一个额外的容器来实施存储信息
- 这道题的信息是从儿子往父亲传的，不需要额外容器
- 如果左儿子和有儿子的返回值分别是p和q，那么就把root往上传；如果左儿子返回值是p而右儿子返回值是NULL，则把左儿子的返回值往上传；要是都是NULL那就传NULL

```c++
class Solution {
public:
    TreeNode* lowestCommonAncestor(TreeNode* root, TreeNode* p, TreeNode* q) {
        if(root == NULL || root == p || root == q){
            return root;
        }

        TreeNode* left = lowestCommonAncestor(root->left, p, q);
        TreeNode* right = lowestCommonAncestor(root->right, p, q);

        if(left && right){
            return root;
        }
        else if(left){
            return left;
        }
        else{
            return right;
        }
    }
};
```
