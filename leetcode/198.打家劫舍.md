#### 动态规划

**dp[i]表示0到i元素能偷的最大数额**

- 如果偷i，i - 1就不能偷，`dp[i] = nums[i] + dp[i - 2]`
- 如果不偷i，i -1 就可以偷`，dp[i] = num[i - 1]`

**边界条件**

由于公式里面有dp[i - 2]，为了避免索引溢出，需要提前处理dp[0]和dp[1]，并且使循环从dp[2]开始



```c++
class Solution {
public:
    int rob(vector<int>& nums) {
        int n = nums.size();
        int ans = 0;
        if(nums.empty()){
            return 0;
        }
        if(nums.size() == 1){
            return nums[0];
        }
        vector<int> dp(n, 0);
        dp[0] = nums[0];
        dp[1] = max(nums[0], nums[1]);
        ans = dp[1];


        //由于公式里面有dp[i - 2]，所以i只能从2开始
        for(int i = 2; i < n; i++){
            dp[i] = max(nums[i] + dp[i - 2], dp[i - 1]);
            ans = max(ans, dp[i]);
        }
        return ans;
    }
};
```
