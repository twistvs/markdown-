#### floyd判圈算法

1. 让slow和fast同时从head出发，slow一次走一步，fast一次走两步，如果有圈，他们一定会在圈内某一点相遇
2. 让slow重新回到head，fast保持位置不变，同时一次走一步，他们再次相遇的位置就一定是圈的起点

```c++
class Solution {
public:
    ListNode* detectCycle(ListNode* head) {
        ListNode* slow = head;
        ListNode* fast = head;
        if (head == NULL || head->next == NULL)
            return NULL;
        while(slow && fast && fast->next){
            slow = slow->next;
            fast = fast->next->next;
            if(slow == fast){
                break;
            }
        }
        if(slow != fast) return NULL;
        
        slow = head;

        while(slow != fast){
            slow = slow->next;
            fast = fast->next;
        }

        return slow;
    }
};
```
