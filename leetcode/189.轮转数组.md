#### 特殊题

- 为了使空间复杂度为1，需要用三反转法
- 向右轮转k个位置，相当于把最后的k%n个元素挪到最前面来

1. 先把整个数组反转
2. 然后把前k%n个元素反转
3. 把后面的元素反转

```c++
class Solution {
public:
    void rotate(vector<int>& nums, int k) {
        int n = nums.size();
        k %= n; // 处理 k > n 的情况
        if (k == 0) return;

        // 1. 全局反转
        reverse(nums.begin(), nums.end());

        // 2. 反转前 k 个元素
        // nums.begin() + k 指向的是下标为 k 的元素
        // reverse 会处理 [0, k-1] 范围
        reverse(nums.begin(), nums.begin() + k);

        // 3. 反转后面剩余的元素
        // 范围是 [k, n-1]
        reverse(nums.begin() + k, nums.end());
    }
};
```
