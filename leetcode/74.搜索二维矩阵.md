#### 与搜索二维矩阵Ⅱ的区别

- 在搜索二维矩阵Ⅱ中，没有**每一行的第一个数大于上一行最后一个数**的条件，矩阵并不是严格递增的，所以只能使用从右上角一步一步收缩的方法，时间复杂度为O(m + n)
- 在这道题中，依旧可以用上述方法，但是这道题额外带来的严格递增的条件，使二分查找变得可用，而二分查找的时间复杂度为O(logmn)，要更快一些

------

#### 两次二分查找

1. 第一次二分查找在第一列里面找，找到比target小的最小值，然后用这一行进行第二次二分查找
2. 第二次二分查找在上述行中查找，如果查找不到则返回false

```c++
class Solution {
public:
    bool searchMatrix(vector<vector<int>>& matrix, int target) {
        int m = matrix.size();
        int n = matrix[0].size();
        //如果target大于最大的或者小于最小的，则返回false
        if(target < matrix[0][0] || target > matrix[m - 1][n - 1]) return false;

        int left = 0;
        int right = m - 1;

        //第一次二分找到第一列中小于target的最大值，即最后的right
        while(left <= right){
            int mid = (left + right)/2;

            if(matrix[mid][0] == target){
                return true;
            }
            else if(target > matrix[mid][0]){
                left = mid + 1;
            }
            else{
                right = mid - 1;
            }
        }

        int row = right;

        left = 0;
        right = n - 1;
        //第二次二分在right所在的行
        while(left <= right){
            int mid = (left + right)/2;

            if(matrix[row][mid] == target){
                return true;
            }
            else if(target > matrix[row][mid]){
                left = mid + 1;
            }
            else{
                right = mid - 1;
            }
        }

        //如果二分结束了都没有找到，就返回false
        return false;
    }
};
```
