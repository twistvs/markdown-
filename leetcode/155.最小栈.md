#### 辅助栈

> [!CAUTION]
>
> ​	仅用一个`int minIndex`是不可行的，因为每次最小值出栈后，都需要设置新的最小值，如果只用minIndex，则每次最小值出栈后都需要在遍历一次栈来寻找新的最小值

为了记录最小值的更迭顺序，需要用辅助栈来存放最小值的更迭



1. 用vector来作为最小栈，因为vector拥有检索功能，可以O(1)检索最小值
2. 用stack来作为辅助栈，每当有新的最小值出现，就把他的vector下标存进来

```c++
class MinStack {
public:
    vector<int> stk;
    stack<int> minIndex;

    MinStack() {
        
    }
    
    void push(int val) {
        stk.push_back(val);
        if(minIndex.empty()){
            minIndex.push(0);
        }
        //如果新入栈的值比当前最小值更小，就把他的下标插到minIndex中
        else{
            if(val < stk[minIndex.top()]){
                minIndex.push(stk.size() - 1);
            }
        }
    }
    
    void pop() {
        //如果当前的最小值出栈了，就把minIndex的头弹出
        if(minIndex.top() == stk.size() - 1){
            minIndex.pop();
        }
        stk.pop_back();
    }
    
    int top() {
        return stk.back();
    }
    
    int getMin() {
        return stk[minIndex.top()];
    }
};
```
