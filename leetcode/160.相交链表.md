#### 特殊解法

1. 从两个链表的起点开始，一起往后next
2. 如果A链表走到了NULL，则把node1放到B链表的头；反之亦然

如果两链表是相交的，则node1与node2必然会相遇；如果不相交，则node1与node2会同时变成NULL

```c++
class Solution {
public:
    ListNode *getIntersectionNode(ListNode *headA, ListNode *headB) {
        ListNode* node1 = headA;
        ListNode* node2 = headB;

        while(true){
            if(!node1 && !node2){
                return NULL;
            }
            if(node1 == node2){
                return node1;
            }
            if(!node1){
                node1 = headB;
                continue;
            }
            if(!node2){
                node2 = headA;
                continue;
            }
            node1 = node1->next;
            node2 = node2->next;
        }

        return NULL;
    }
};
```
