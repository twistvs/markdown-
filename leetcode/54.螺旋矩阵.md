#### 边界收缩

- 每一次转向都代表着一行/一列的元素被完全遍历完（而且总是矩阵剩余的元素中最外层的），相当于矩阵收缩了

```c++
class Solution {
public:
    vector<int> spiralOrder(vector<vector<int>>& matrix) {
        int m = matrix.size();
        int n = matrix[0].size();

        vector<int> ans;

        int up = 0, down = m - 1, left = 0, right = n - 1;

        //每次转向的时候，都需要收缩边界，如果收缩完边界导致up > down或者left > right，那么循环就结束，所有元素都遍历完了
        while(true){
            for(int j = left; j <= right; j++){
                ans.push_back(matrix[up][j]);
            }
            up++;
            if(up > down) break;

            for(int i = up; i <= down; i++){
                ans.push_back(matrix[i][right]);
            }
            right--;
            if(right < left) break;

            for(int j = right; j >= left; j--){
                ans.push_back(matrix[down][j]);
            }
            down--;
            if(down < up) break;

            for(int i = down; i >= up; i--){
                ans.push_back(matrix[i][left]);
            }
            left++;
            if(left > right) break;
        }

        return ans;
    }
};
```
