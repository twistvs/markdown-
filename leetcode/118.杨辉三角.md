#### 动态规划+数学

- 除了每行最左边和最右边的元素固定为1之外，其他元素都满足`ans[i][j] = ans[i - 1][j - 1] + ans[i - 1][j]`

- 第一行和第二行固定为1

```c++
class Solution {
public:
    vector<vector<int>> generate(int numRows) {
        vector<vector<int>> ret(numRows);
        for (int i = 0; i < numRows; ++i) {
            //通过resize来动态改变每行的长度，要比push_back的性能好很多；因为vector的随机访问性能要比插入删除好得多
            ret[i].resize(i + 1);
            ret[i][0] = ret[i][i] = 1;
            for (int j = 1; j < i; ++j) {
                ret[i][j] = ret[i - 1][j] + ret[i - 1][j - 1];
            }
        }
        return ret;
    }
};

```
