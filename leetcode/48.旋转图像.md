- 旋转图像90度可以分解为两个步骤：

1. 将矩阵沿**主对角线**（只有主对角线的对称元素的i和j是相反的，副对角线不是）反转；注意反转的时候智能遍历对角线一边的元素，否则会导致翻转两次又给翻转回去了
2. 将每一行进行reverse

```c++
class Solution {
public:
    void rotate(vector<vector<int>>& matrix) {
        int n = matrix.size();

        // 1. 沿主对角线翻转 (Transpose)
        for (int i = 0; i < n; i++) {
            // 注意 j 从 i+1 开始，避免重复翻转变回原样
            for (int j = i + 1; j < n; j++) {
                swap(matrix[i][j], matrix[j][i]);
            }
        }

        // 2. 每一行进行左右反转
        for (int i = 0; i < n; i++) {
            // 直接利用我们之前讨论过的 std::reverse 函数
            reverse(matrix[i].begin(), matrix[i].end());
        }
    }
};
```
