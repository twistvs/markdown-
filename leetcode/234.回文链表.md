#### 快慢指针找链表中点+链表反转

##### 快慢指针找链表中点

- while的判定条件是`fast->next && fast->next->next`

1. 对于节点数量为奇数的链表，循环结束后，slow正好处在最中间的节点
2. 对于节点数量为偶数的链表，循环结束后，slow处在中线左边的节点

```c++
ListNode* slow = head;
ListNode* fast = head;

while(fast->next && fast->next->next){
    slow = slow->next;
    fast = fast->next->next;
}
```

##### 链表反转

- 三指针反转法，让`curr->next = prev`，用nextNode标记curr右边的节点，但是不用他来连接

```c++
    ListNode* reserveList(ListNode* head){
        ListNode* curr = head;
        ListNode* prev = NULL;
        
        while(curr){
            //这句一定要写在curr = nextNode的前面，因为如果nextNode是NULL，curr也等于NULL，则curr->next就会导致报错
            ListNode* nextNode = curr->next;
            curr->next = prev;
            prev = curr;
            curr = nextNode;
        }
        return prev;
    }
```

------

- 对于这道题，我们要找到右半部分的起点，这个起点是`slow->next`

- 如果链表节点数是奇数，则右半部分会比左半部分少一个节点，所以在循环比较的过程中，只要右半部分比较完了，就说明链表是回文的

```c++
class Solution {
public:
    bool isPalindrome(ListNode* head) {
        //快慢指针找链表中点
        ListNode* slow = head;
        ListNode* fast = head;

        while(fast->next && fast->next->next){
            slow = slow->next;
            fast = fast->next->next;
        }

        ListNode* firstHalf = head;
        ListNode* secondHalf = reserveList(slow->next);
        bool ans = true;

        while(secondHalf){
            if(firstHalf->val == secondHalf->val){
                firstHalf = firstHalf->next;
                secondHalf = secondHalf->next;
            }
            else{
                ans = false;
                break;
            }
        }
        return ans;
    }

    //反转链表函数
    ListNode* reserveList(ListNode* head){
        ListNode* curr = head;
        ListNode* prev = NULL;
        
        while(curr){
            ListNode* nextNode = curr->next;
            curr->next = prev;
            prev = curr;
            curr = nextNode;
        }
        return prev;
    }
};
```
