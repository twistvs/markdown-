#### 双指针

- 思路很简单，slow和fast相隔n，当fast到达链表尾部的时候，slow到达倒数第n个结点
- **但是**，这个题有一个特殊的边界条件

<img src="C:\Users\lee\AppData\Roaming\Typora\typora-user-images\image-20260324161852376.png" alt="image-20260324161852376" style="zoom:50%;" />

当查找的时倒数第size个（size是链表的长度）结点时，fast会直接初始化到NULL上，此时后面的fast->next判定就会报错，所以需要单独判断这种情况

```c++
class Solution {
public:
    ListNode* removeNthFromEnd(ListNode* head, int n) {
        ListNode* slow = head;
        ListNode* fast = head;
        //fast和slow相隔n
        for(int i = 0; i < n; i++){
            fast = fast->next;
        }
		
        //判断边界条件，此时删除的是第一个结点，直接把head后移一位即可
        if(fast == NULL){
            head = slow->next;
        }
        //其他情况下，正常使用双指针来寻找倒数第n个结点的位置
        else{
            while(fast->next){
                slow = slow->next;
                fast = fast->next;
            }
            //如果让slow正好停在倒数第n个结点上，slow无法获得前一个结点，所以就没法删除；所以需要让slow停在倒数第n个结点的前一个结点
            slow->next = slow->next->next;
        }
        return head;
    }
};
```

#### （优化）利用哑节点无视特殊情况

- 如果在前面加上哑节点，即便是只有一个节点的情况依旧不需要特殊讨论

```c++
class Solution {
public:
    ListNode* removeNthFromEnd(ListNode* head, int n) {
        ListNode* pre = new ListNode(0);
        pre->next = head;
        ListNode* slow = pre;
        ListNode* fast = pre;

        for(int i = 0; i < n; i++){
            fast = fast->next;
        }

        while(fast && fast->next){
            slow = slow->next;
            fast = fast->next;
        }

        slow->next = slow->next->next;

        return pre->next;
    }
};
```
