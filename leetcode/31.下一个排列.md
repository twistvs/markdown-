#### 特殊解法

1. 从右往左找第一个`nums[i] < nums[i + 1]`
2. 如果`i == -1`，说明nums是纯降序，也就是最后一个排列，那么他的下一个排列就是把整个nums颠倒reverse，回到第一个排列
3. 如果`i != -1`，则从右往左找第一个大于`num[i]`的`nums[j]`
4. `swap(nums[i], nums[j])`，将i后面的部分（不包括i）reverse

```c++
class Solution {
public:
    void nextPermutation(vector<int>& nums) {
        int n = nums.size();

        int i;
        for (i = n - 2; i >= 0; i--) {
            if (nums[i] < nums[i + 1]) {
                break;
            }
        }

        //只有i >= 0时才需要找j，否则之间把nums全部reverse
        if (i >= 0) {
            int j;
            for (j = n - 1; j >= i; j--) {
                if (nums[j] > nums[i]) {
                    break;
                }
            }

            swap(nums[i], nums[j]);
        }
		
		//当i == -1时，相当于reverse(nums.begin(), nums.end())
        reverse(nums.begin() + i + 1, nums.end());
    }
};
```
